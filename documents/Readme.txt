AnimeKakkoi es un aplicación Open Source.

Se permite la distribución e instalación de un mismo instalador en diferentes terminales.


-Los requetimientos para la ejecución están supuestos a ser los mínimos para cualquier terminal. De igual forma, aún siendo tú terminal basada en Microsoft Windows de arquitectura x86 - x64 bits, puedes escribirnos indicándonos las especificaciones de tú sistema.

- Para revisar las mejores y cambios en las nuevas versiones: dirigete al blog y busca la entrada correspondiente con el número de release recien instalado.



Dirección: corderoski.wordpress.com/bluegletek-soft/animekakkoi/
