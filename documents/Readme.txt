AnimeKakkoi v0.1.2 – BluegleTek Soft 


AnimeKakkoi es una aplicación Open Source.
Se permite su distribución e instalación en número ilimitado de terminales.


Requisitos / Requerimientos:
- Sistema Operativo Microsoft Windows de arquitectura x86 / x64
- Framework .NET 3.5



	# Release Notes # 

	- R 0.1.2

- Agregada las opciones para uso de proxy.
- Configuración y corrección de idioma preferido por usuario.
- Agregada función de eliminar usuarios.
- Agregadas configuraciones de estilos de fuentes y colores.
- Agregado formulario de preferencias.
- AGREGADO nueva fuente externa MyAnimeList
